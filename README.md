
# Minimal implementation of the Graphics2D class that produces
a SVG Document. 

This class only supports the following methods:

<ul>
<li>draw</li>
<li>fill</li>
<li>drawGlyphVector</li>
<li>drawString</li>
<li>setColor</li>
<li>getColor</li>
<li>setFont</li>
<li>getFont</li>
<li>getStroke</li>
<li>setStroke</li>
</ul>
